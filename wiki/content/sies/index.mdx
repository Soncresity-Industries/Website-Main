## About SI: Essentials
**SI: Essentials** is a powerful library mod that not only adds useful tools for developrs but also has useful submodules that players can use.

## Environment
**SI: Essentials** is required on both the client and the server side, but mods that depend on **SI: Essentials** may still be server-side only!
You can find the links to the mod in the `Open` dropdown at the top of the page.

## Compatibility
The latest Version of **SI: Essentials** is compatible with [NeoForge](https://neoforged.net) and [Fabric](https://fabricmc.net) 1.21.8, 1.21.10 and 1.21.11.
