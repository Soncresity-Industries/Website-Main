The Registry Utilities class makes the process of registering Items and Blocks easier by providing useful helper methods.

# Class
File: `RegistryUtils.java`  
Package: `dev.soncresityindustries.es.api.util`

# Methods
## `registerBlock`
```java tab="sies >=2.2.0-fabric"
public static Block registerBlock(String name, Function<BlockBehaviour.Properties, Block> function, boolean registerBlockItem, boolean fireResistant, boolean explosionResistant) {}
```

### Usage [!toc]
```java tab="sies >=2.2.0-fabric"
public static final Block MY_BLOCK = RegistryUtils.registerBlock("my_block", props -> new Block(props), registerBlockItem, fireResistant, explosionResistant);
```

## `registerBlockItem`
<Callout title="Warning" type="warn">

```java tab="sies >=2.2.0-fabric"
public static void registerBlockItem(String name, Block block, boolean fireResistant, boolean explosionResistant) {}
```

```java tab="sies >=2.2.0-fabric"
RegistryUtils.registerBlockItem("my_block", block, fireResistant, explosionResistant);
```
