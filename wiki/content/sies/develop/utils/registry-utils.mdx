<Callout title="Introduction" type="info">
  A fully functional Version of this class has first been introduced in Version `2.2.0`
</Callout>

<Callout title="Warning" type="warn">
  This Utility class is currently **only available for `fabric`!**
</Callout>

The Registry Utilities class makes the process of registering Items and Blocks easier by providing useful helper methods.

<Callout title="Warning" type="warn">
  An error was made in versions below 2.2.2 where you could only use the RegistryUtils to register items and blocks to SI: Essentials itself and not to your own mod.  
  This has been fixed in Version 2.2.2 by introducing a new `modid` argument to each Method.
</Callout>

# Class
File: `RegistryUtils.java`  
Package: `dev.soncresityindustries.es.api.util`

### Usage [!toc]
