<Callout title="Introduction" type="info">
  A fully functional Version of this class has first been introduced in Version `2.1.0`
</Callout>

The Item Utilities class allows you parse Items and Item Stacks from Strings or vice versa.

# Class
File: `ItemUtils.java`  
Package: `dev.soncresityindustries.es.api.util`

# Methods
## `getItemOrDefault`
```java tab="sies >=2.1.0"
public static Item getItemOrDefault(String itemID, @NotNull Item defaultItem) {}
```
Returns the `defaultItem` value if the method runs into an error (e.g. `itemID` does not match an Item in Minecraft) otherwise returns the parsed `Item` from the `itemID`.

### Usage [!toc]
```java tab="sies >=2.1.0"
Item item = ItemUtils.getItemOrDefault(itemID, defaultItem);
```

## `getItem` (from `String`)
```java tab="sies >=2.1.0"
public static Item getItem(String itemID) {}
```
Returns the parsed `Item` from the `itemID`.
Throws `NoSuchElementException` if the item can't be parsed.

### Usage [!toc]
```java tab="sies >=2.1.0"
Item item = ItemUtils.getItem(itemID, defaultItem);
```


</Callout>

## `getItemStackOrDefault`
```java tab="sies >=2.1.0"
public static ItemStack getItemStackOrDefault(String itemID, @NotNull ItemStack defaultItemStack) {}
```
Returns the `defaultItemStack` value if the method runs into an error (e.g. `itemID` does not match an Item in Minecraft) otherwise returns the parsed `ItemStack` with the **amount of 1**.
Refer to [`getItemStackOrDefault` (with `count`)](#getitemstackordefault-with-count) to customize the returned amount of items in the `ItemStack`.

### Usage [!toc]
```java tab="sies >=2.1.0"
ItemStack itemStack = ItemUtils.getItemStackOrDefault(itemID, defaultItemStack);
```
