<Callout title="Introduction" type="info">
  A fully functional Version of this class has first been introduced in Version `2.1.0`
</Callout>

The Message Utilities class allows you to send (privately and broadcast) messages (in the chat or Action Bar) with the [Adventure/MiniMessage API](https://docs.papermc.io/adventure/minimessage/format/).

# Class
File: `MessageUtils.java`
Package: `dev.soncresityindustries.es.api.util`

# Methods
## `sendClientMessage`
```java tab="sies >=2.1.0"
public static void sendClientMessage(String prefix, ServerPlayer player, String serializedMessage, boolean isActionBar) {}
```
Sends a messasge privately to the specified `ServerPlayer`.
<Callout title="Tip" type="idea">
  If you don't want to use a prefix in the message, you can leave it empty (`""`).
</Callout>

### Usage [!toc]
```java tab="sies >=2.1.0"
MessageUtils.sendClientMessage(prefix, player, serializedMessage, isActionBar);
```

## `getDeserializedText` (with `prefix`)
```java tab="sies >=2.1.0"
public static net.minecraft.network.chat.Component getDeserializedText(String prefix, String serializedMessage) {}
```
Transforms a serialized text and prefix with MiniMessage formatting into a Vanilla Chat Component while preserving its format.
<Callout title="Tip" type="idea">
  If you don't want to use a prefix in the message, you can leave it empty (`""`) or use the [`getDeserializedText`](#getdeserializedtext) method.
</Callout>

### Usage [!toc]
```java tab="sies >=2.1.0"
net.minecraft.network.chat.Component component = MessageUtils.getDeserializedText(prefix, serializedMessage);
```

## `getDeserializedText`
```java tab="sies >=2.1.0"
public static net.minecraft.network.chat.Component getDeserializedText(String prefix, String serializedMessage) {}
```
Transforms a serialized text with MiniMessage formatting into a Vanilla Chat Component while preserving its format.

### Usage [!toc]
```java tab="sies >=2.1.0"
net.minecraft.network.chat.Component component = MessageUtils.getDeserializedText(serializedMessage);
```

## `broadcastMessage` (`ServerPlayer` source)
```java tab="sies >=2.1.0"
public static void broadcastMessage(ServerPlayer source, String serializedMessage, boolean actionBar) {}
```
Broadcasts a serialized message to every player on the server either in the chat (`actionBar = false`) or in the Action bar (`actionBar = true`).
<Callout title="Cannot provide a `ServerPlayer` instance as the source?" type="info">
  Use [`broadcastMessage`](#broadcastmessage-minecraftserver-source) with the `MinecraftServer` source.
</Callout>

### Usage [!toc]
```java tab="sies >=2.1.0"
MessageUtils.broadcastMessage(source, serializedMessage, actionBar);
```
