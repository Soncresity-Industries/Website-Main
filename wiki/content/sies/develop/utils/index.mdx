Utility Classes are classes that you, as a developer, can use to make the development of your mod easier.  
Below are all currently available Classes for you to choose from.  

<Cards>
  <Card icon={<Slash color="#026DCA"/>} href="/sies/develop/utils/command-utils" title="Command Utilities">
    Allows execution of commands as a player
  </Card>
  <Card icon={<DecimalsArrowRight color="#026DCA"/>} href="/sies/develop/utils/decimal-utils" title="Decimal Utilities">
    Allows rounding decimal values to a set number of decimal points
  </Card>
  <Card icon={<Binary color="#026DCA"/>} href="/sies/develop/utils/file-hash-utils" title="File Hash Utilities">
    Allows getting the SHA256 file hash of any given file
  </Card>
  <Card icon={<Box color="#026DCA"/>} href="/sies/develop/utils/item-utils" title="Item Utilities">
    Allows parsing Items and Item Stacks
  </Card>
  <Card icon={<MessageCircleCode color="#026DCA"/>} href="/sies/develop/utils/message-utils" title="Message Utilities">
    Allows sending and deserializing messages with the Adventure/MiniMessage API
  </Card>
  <Card icon={<CirclePlus color="#026DCA"/>} href="/sies/develop/utils/registry-utils" title="Registry Utilities (Fabric)">
    Makes the process of registering Items and Blocks easier (Fabric only!)
  </Card>
  <Card icon={<Globe color="#026DCA"/>} href="/sies/develop/utils/singleplayer-utils" title="Singleplayer Utilities">
    Allows checking whether a player is the host of a Singleplayer World
  </Card>
  <Card icon={<SquareDashed color="#026DCA"/>} href="/sies/develop/utils/slot-utils" title="Slot Utilities">
    Allows getting the x and y position of a slot in a large chest GUI
  </Card>
  <Card icon={<AudioLines color="#026DCA"/>} href="/sies/develop/utils/sound-parser" title="Sound Parser">
    Allows to parse and play sounds privately for a player or broadcast them across a dimension
  </Card>
  <Card icon={<Type color="#026DCA"/>} href="/sies/develop/utils/title-utils" title="Title Utilities">
    Allows you to show and broadcast fully customizable Titles to players
  </Card>
</Cards>
