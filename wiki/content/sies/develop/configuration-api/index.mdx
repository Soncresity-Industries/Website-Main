# Introduction 
Our Configuration API allows you, as a developer, to create multiple configurations in your projects and access and set configuration keys and values.  
Currently, you can choose the `cnfg` or the `objt` file type. ([More Info](/sies/develop/file-formats))  

## Cnfg Type
If you want to use the cnfg file type, you need to choose between the `CLIENT`, `COMMON` and `SERVER` Cnfg types based on your use case. You can read more about Cnfg Types [here](/sies/develop/configuration-api/cnfg-types)
