## About SI: Death Bolt
**SI: Death Bolt** allows you to spawn and configure the spawning and behavior of a lightning bolt upon a player's death.
This mod is especially useful for multiplayer events as it also allows you to set death actions, such as banning players, kicking them, showing a Title to a/all player/s or sending a message to a/all player/s.

## Environment
**SI: Death Bolt** is only required on the server side, but if you want to use it in your own worlds, you will need to install it on the client.
You can find the links to the mod in the `Open` dropdown at the top of the page.

## Compatibility & Performance
The latest Version of **SI: Death Bolt** is compatible with [NeoForge](https://neoforged.net) 1.21.1, 1.21.4, 1.21.8, 1.21.10 and 1.21.11.
Additionally, **SI: Death Bolt** should not impact your games performance at all.
